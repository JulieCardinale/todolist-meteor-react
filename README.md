## Run the app

```bash

# Clone this repository
$ git clone https://github.com/JulieCardinale/todolist-meteor-react.git

# Go into the repository
$ cd todolist-meteor-react

# Install dependencies
$ meteor npm install

# Run the app
$ meteor

```
